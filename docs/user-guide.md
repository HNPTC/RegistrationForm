# 樣板表單系統使用指南

## 簡介

這是一個視覺化的表單配置系統,讓您無需編寫程式碼就能建立和管理活動報名表單。

### 主要特色
- 🎨 **圖形化介面** - 拖曳、點擊即可編輯
- 👀 **即時預覽** - 立即看到表單外觀
- 🔄 **重複使用** - 欄位庫快速建立表單
- 📱 **LINE 整合** - 支援 LIFF 和官方帳號
- 🌐 **雲端部署** - 發佈到 GitHub Pages

---

## 快速開始

### 第一次使用

#### 1. 開啟編輯器
在瀏覽器中開啟:
```
template-system/config-editor.html
```

#### 2. 載入範本
點擊「載入範本」→ 選擇 `default-config.json`

#### 3. 開始編輯
- 修改表單標題
- 調整欄位內容
- 查看右側即時預覽

#### 4. 匯出配置
點擊「匯出 JSON」→ 儲存檔案

#### 5. 交付部署
將 JSON 檔案交給技術人員上傳到 GitHub

## 編輯器介面說明

### 主要區域

```
┌─────────────────────────────────────────┐
│  ① 頂部工具列                            │
├──────────────────┬──────────────────────┤
│  ② 左側編輯區     │  ③ 右側預覽區         │
│                  │                      │
│  [基本設定]       │  [表單即時預覽]       │
│  [欄位管理]       │                      │
│  [進階設定]       │                      │
│                  │                      │
└──────────────────┴──────────────────────┘
```

#### ① 頂部工具列
- **載入範本** - 開啟現有配置檔
- **匯入 JSON** - 上傳 JSON 檔案
- **匯出 JSON** - 下載配置檔
- **重設** - 清空所有設定

#### ② 左側編輯區
- **基本設定** - 表單標題、LIFF ID 等
- **橫幅設定** - 圖片網址、顯示開關
- **資訊區塊** - 活動內容、地點說明
- **欄位管理** - 新增、編輯、排序欄位

#### ③ 右側預覽區
- 即時顯示表單外觀
- 與實際表單完全相同
- 可測試互動效果

- 寬度: 500-800px
- 高度: 200-400px
- 格式: JPG, PNG, WebP

## 欄位管理

### 欄位類型

#### 1. 文字輸入 (text)
用於: 姓名、備註等
```
設定項目:
- 欄位 ID
- 標題
- 佔位符 (placeholder)
- 是否必填
```

#### 2. 單選 (radio)
用於: 性別、場次等
```
設定項目:
- 欄位 ID
- 標題
- 選項列表
- 排列方式 (垂直/水平)
- 是否必填
```

#### 3. 複選 (checkbox)
用於: 從哪得知、興趣等
```
設定項目:
- 欄位 ID
- 標題
- 選項列表
- 是否必填
```

#### 4. Email 輸入 (email)
用於: Email 地址
```
設定項目:
- 欄位 ID
- 標題
- 佔位符
- 是否必填
```

#### 5. 提醒設定 (remind-section)
用於: 活動提醒功能
```
設定項目:
- 啟用 Line 提醒
- 啟用 Email 提醒
```

---

### 欄位操作

#### 新增欄位
1. 點擊「+ 新增欄位」
2. 選擇欄位類型
3. 填寫欄位設定
4. 點擊「確認」

#### 編輯欄位
1. 點擊欄位右側的「編輯」按鈕
2. 修改設定
3. 點擊「儲存」

#### 刪除欄位
1. 點擊欄位右側的「刪除」按鈕
2. 確認刪除

#### 調整順序
1. 拖曳欄位左側的「☰」圖示
2. 移動到目標位置
3. 放開滑鼠

#### 開關欄位
1. 點擊欄位右側的切換按鈕
2. 停用的欄位不會顯示在表單中
3. 但配置會保留,可隨時重新啟用

## 使用欄位庫

### 什麼是欄位庫?
預先定義的常用欄位,可快速新增到表單中。

### 可用欄位

#### 基本資訊
- **姓名** - 文字輸入,必填
- **性別** - 單選 (男/女),必填
- **聯絡電話** - 文字輸入,必填
- **Email** - Email 輸入,選填

#### 人口統計
- **年齡區段** - 單選,5個年齡層

#### 行銷
- **從哪得知** - 複選,4個來源選項

### 使用方式
1. 點擊「從欄位庫新增」
2. 瀏覽可用欄位
3. 勾選需要的欄位
4. 點擊「新增到表單」

## 匯入與匯出

### 匯出 JSON
1. 完成編輯後點擊「匯出 JSON」
2. 輸入檔案名稱 (例: `event-2026-02`)
3. 點擊「下載」
4. 檔案會儲存到下載資料夾

### 匯入 JSON
1. 點擊「匯入 JSON」
2. 選擇本地的 JSON 檔案
3. 配置會自動載入到編輯器
4. 可繼續編輯或直接使用

## 實際案例

### 案例 1: 建立健康講座報名表

#### 需求:
- 3個場次可選
- 收集姓名、性別、年齡
- 詢問從哪得知活動
- 提供 Line 提醒

#### 步驟:
1. 載入 `default-config.json`
2. 修改表單標題為「健康講座報名表」
3. 編輯「報名場次」選項:
   - 2/10 (一) 14:00 - 16:00
   - 2/15 (六) 10:00 - 12:00
   - 2/20 (四) 19:00 - 21:00
4. 保留姓名、性別、年齡欄位
5. 保留「從哪得知」和「提醒設定」
6. 匯出為 `health-lecture-2026-02.json`

---

### 案例 2: 建立簡易問卷

#### 需求:
- 只收集基本資料
- 不需要場次選擇
- 不需要提醒功能

#### 步驟:
1. 載入 `default-config.json`
2. 修改表單標題為「會員資料調查」
3. 停用「報名場次」欄位
4. 停用「提醒設定」欄位
5. 從欄位庫新增「聯絡電話」
6. 匯出為 `member-survey.json`

---

### 案例 3: 複製上次活動表單

#### 需求:
- 使用上次活動的表單結構
- 只修改場次時間和橫幅圖片

#### 步驟:
1. 匯入 `event-2026-01.json`
2. 更新橫幅圖片網址
3. 修改「報名場次」的日期和時間
4. 其他欄位保持不變
5. 匯出為 `event-2026-02.json`

## 常見問題

### Q: 我不小心刪除了欄位,怎麼辦?
**A:** 
- 如果還沒匯出,點擊「重設」重新載入
- 如果已匯出,重新匯入上一個版本的 JSON

### Q: 預覽區沒有更新?
**A:** 
- 檢查是否有錯誤訊息
- 嘗試重新整理頁面
- 確認 JSON 格式正確

### Q: 如何新增更多場次選項?
**A:** 
1. 點擊「報名場次」欄位的「編輯」
2. 在選項列表中新增項目
3. 點擊「儲存」

### Q: 可以修改欄位的 ID 嗎?
**A:** 
- 不建議修改,除非您了解技術細節
- ID 用於資料提交,修改可能導致資料遺失

### Q: 如何確認配置正確?
**A:** 
1. 查看右側預覽區
2. 匯出 JSON 後使用 JSON 驗證工具檢查
3. 請技術人員協助測試

## 進階技巧

### 技巧 1: 建立欄位範本
將常用的欄位組合儲存為獨立的 JSON 檔案,方便重複使用。

### 技巧 2: 使用描述性 ID
為欄位使用有意義的 ID,例如:
- `realName` 而非 `field1`
- `sessionTime` 而非 `q1`

### 技巧 3: 保持配置簡潔
- 停用不需要的欄位而非刪除
- 未來可能會再次使用

### 技巧 4: 定期備份
將 JSON 檔案備份到雲端硬碟或 Email 給自己。
